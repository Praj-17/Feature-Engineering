# The Processes to handle all modules in an ML project

1. EDA (Exploratory Data Analysis)
2. Feature Engineering 
    a. Missing Values (checking, handling, removing/imputing)
    b. Handling Imbalanced Dataset
    c. Outliers (checking, handling, removing/imputing)
    d. Converting Categorical to Numerical Features
    e. Transformation (Scaling, standardization, Normalization)
3. Feature Selection
    a. Correlations
    b. Chi-Square, KNN Tests
    c. Genetic Algorithm 
    d. Feature Importance
4. Model Selection and ModelBuilding
5. Hyper-Parameter Tuning 
6. Model Deployment
